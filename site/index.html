Описание проекта

Данный отчет по курсовому проекту выполнен в соответствии с заданием по дисциплине «Инженерное проектирование».

Темой проекта является разработка веб-страниц для сайта по продаже недвижимости и вывод данных из базы.

В данной работе рассматривается создание удобных интерфейса и функционала сайта по продаже недвижимости, который будет включать в себя контроль учетных записей пользователей со стороны администратора, поддержание актуальной информации об услугах, агентствах, риелторах, владельцев недвижимости, покупателях и самой недвижимости. Система создана для автоматизации процессов покупки/продажи/сдачи в аренду жилья, для облегчения работы администраторов по наполнению и редактированию контента на сайте, для облегчения построения деловых отношений между владельцами недвижимости, потенциальными покупателями и риэлторами.

Целью проекта является разработка автоматизированной системы, которая будет включать в себя базу данных и веб-сервис, обладающее интерфейсом просмотра и редактирования, а также средствами обработки хранящейся в базе данных информации.

Актуальностью данной темы является современный динамично развивающийся мир, открывающий новые перспективы для роста некоторых сфер экономической деятельности. Одной из таких сфер является сфера продаж. Развитие данной отрасли обусловлено множеством различных факторов. Для более эффективного развития необходимо использовать автоматизированные информационные системы.

Задачи по проекту:

  • Проанализировать аналогичные сервисы
  • Спроектировать сайт в исследуемой предметной области
  • Разработать и протестировать проект

Стек технологий, используемый в проекте

Angular

Фреймворк от компании Google для создания клиентских приложений. Прежде всего он нацелен на разработку SPA-решений (Single Page Application), то есть одностраничных приложений. Angular предоставляет такую функциональность, как двустороннее связывание, позволяющее динамически изменять данные в одном месте интерфейса при изменении данных модели в другом, шаблоны, маршрутизация и так далее. Одной из ключевых особенностей Angular является то, что он использует в качестве языка программирования TypeScript.

Node.js

Среда выполнения кода на JavaScript, которая построена на основе движка JavaScript Chrome V8, который позволяет транслировать вызовы на языке JavaScript в машинный код. Node.js прежде всего предназначен для создания серверных приложений на языке JavaScript.

MySQL

Cистема управления реляционными базами данных (СУБД). На сегодняшний день это одна из самых популярных систем управления базами данных.

Git

Распределённая система управления версиями

Draw.io

Бесплатное приложение, предназначенное для моделирования диаграмм и блок-схем бизнес-процессов.

Figma

Онлайн-сервис для разработки интерфейсов и прототипирования с возможностью организации совместной работы в режиме реального времени. Макет разрабатываемых страниц проекта выполняется в данном онлайн-сервисе.
